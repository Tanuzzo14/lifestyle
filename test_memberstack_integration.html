How It Works

1. User Registration/Login

When a user logs in, the system automatically checks Memberstack subscription status.

2. Trial Period

All users get 6 months (180 days) free access from their registration date.

During trial: isInTrial = true

3. Active Subscription

Users with active plans in Memberstack can access all features.

With subscription: isActive = true

4. Expired State

When trial expires and no active subscription exists, users see a courtesy page.

The page offers Memberstack modals to view pricing or manage account.
